# Notes to self:

## To browse locally

```shell
docker-compose up
```
Browse [localhost](http://localhost)

## Deploy to firebase

```shell
npm install
node node_modules/firebase-tools/bin/firebase login
node node_modules/firebase-tools/bin/firebase init
node node_modules/firebase-tools/bin/firebase deploy
```
